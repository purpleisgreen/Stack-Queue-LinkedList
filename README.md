# Stack, Queue, and Linked List
This features custom made Stack, Queue and Linked List Classes. 

The purpose was to explore different linear data structures. Object of type StudentInfo was used as the data. 

### Stack 
Includes the following features: push, pop, peek and display.

### Queue
Includes the following features: enqueue, dequeue, peek and display.

### Linked List
Includes the following features: add to front, add to rear, remove from certain index, add from certain index.
